# 🏥 Hospital Management System

A modern web-based Hospital Management System built with a clean UI and secure authentication. This project helps manage hospital operations efficiently with a scalable architecture.

---

## 🚀 Features

### 🔐 Authentication
- Secure login system
- Username & password validation
- Clean and minimal UI
- "Forgot Password" support

### 🖥️ User Interface
- Responsive design
- Simple and intuitive login experience
- Modern layout with centered form
- Clean typography and spacing

### ⚙️ Backend (Expected / Extendable)
- REST APIs for authentication
- Role-based access (Admin, Doctor, Staff)
- Secure session/token handling
